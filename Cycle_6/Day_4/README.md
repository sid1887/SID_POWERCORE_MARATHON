## ðŸ§  Key Learnings
- Arduino boards differ mainly in architecture, voltage, and I/O
- Clone boards are functionally equivalent for learning
- AVR is simple and educational; ARM is modern and powerful
- Voltage mismatch (5V vs 3.3V) can permanently damage hardware
- GPIO pins have strict current and voltage limits
- Arduino and ESP32 use SAR ADCs with different quality levels
- PWM is digital switching, not true analog output
- UART, I2C, and SPI serve different communication needs
- ESP32 provides performance and connectivity at the cost of complexity
- Clock accuracy affects timing, communication, and correctness
- Flash stores code, RAM stores runtime data, EEPROM stores small persistent data
